# 🤝 PureHope – Charity Donation Platform

## 📌 Project Overview

PureHope is a charity donation platform designed to connect donors with meaningful causes and provide support to underprivileged communities. The platform enables users to contribute towards various social causes, helping create a better future for humans, animals, education, healthcare, and environmental initiatives. Every contribution made through PureHope aims to bring positive change and support those in need.

## ⚙️ Technologies Used

1. Python
2. Flask (Web Framework)
3. HTML
4. CSS

## 🌐 Features

- Donation Platform for Multiple Social Causes
- Support for Human Welfare and Animal Care
- User-Friendly Interface
- Cause-Based Donation Categories
- Secure and Simple Navigation
- Responsive Web Design
- Encourages Social Impact and Community Support
